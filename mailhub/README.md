# Mail Hub

This is a self hosted dockerized mail server supporting scanning with clamav, spamassassin, and amavisd-new.

Here are the packages used in my custom server setup:

clamav
mysql - I think for spamassassin,
amavisd-new
procmail
spamassassin
postfix
courier-imap
fetchmail

I would like to replace courier-imap with dovecot + sieve support.

PetHub frontend: React 18 + Vite SPA served by nginx. The browser talks to `/api/*` on the same origin; nginx proxies those requests to the Flask backend. Session authentication uses Flask-Login cookies with `axios` (`withCredentials: true`). Dev workflow: `npm run dev` in `pethub/frontend` on port 3002 with Vite proxying `/api` to the backend; set `CORS_ORIGINS` on Flask to include `http://localhost:3002`.
